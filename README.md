# **EmporiUm Sales Analysis**



For this capstone project, I have been assigned to analyze **Texas** sales territory. This project shows findings for in-store sales of that territory and answers questions asked by the Sales Manager. The SQL script in this repository displays that I calculated the total revenue, month-to-month revenue breakdown, comparison of total revenues based on categories and store rankings in the state. The results have been used to create Pivot Charts to support sales recommendation.
